**PHP Masterclass** es una plataforma educativa de élite diseñada para transformar desarrolladores en arquitectos de backend. A través de un recorrido de 21 módulos técnicos, exploramos desde los cimientos del lenguaje hasta la construcción de sistemas ERP complejos y APIs REST robustas.

---

## 🏛️ Arquitectura del Proyecto

Este no es un sitio web común. Se ha diseñado siguiendo principios de **Máximo Rendimiento y Simplicidad**:

- **Arquitectura 100% Estática**: Sin dependencias de bases de datos pesadas ni frameworks SPA complejos. Rápido, seguro y compatible con cualquier entorno (Netlify, GitHub Pages, Local).
- **Diseño Cinemático Bento**: Una interfaz visualmente impactante inspirada en las mejores documentaciones modernas.
- **Sistema de Componentes Modular**: Navegación y menús centralizados vía JavaScript puro para un mantenimiento ágil.
- **Deep Black Aesthetics**: Estética técnica de alta fidelidad optimizada para la legibilidad del código.

---

## 🗺️ Mapa de Ruta (21 Niveles)

El curso se divide en 21 hitos fundamentales:

## 🚀 Cómo Empezar

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/ByChokeYT/curso_de_php.git
   ```
2. **Ejecuta el servidor interno de PHP** (dentro de la carpeta raíz):
   ```bash
   php -S localhost:8000
   ```
3. **Abre tu navegador** en `http://localhost:8000`.
